# Leaflyx — Next.js + Tailwind (MVP)

## Quickstart
```bash
npm i
npm run dev
# open http://localhost:3000
```

## Routes
- `/` — Home
- `/products` — Shop
- `/products/[slug]` — Product detail
- `/about` — About
- `/faq` — FAQ
- `/cart` — Cart

### Notes
- Checkout is a placeholder. Use a provider that explicitly permits hemp/THCA.
- Logo file lives at `public/leaflyx_logo.png` (replace with your own when ready).
